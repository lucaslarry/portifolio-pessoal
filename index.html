Projetos

Ouvidoria em Python

O projeto de Ouvidoria em Python é uma aplicação para registrar reclamações, elogios e críticas. Os usuários podem visualizar, apagar e atualizar os comentários. A aplicação possui conexão com um banco de dados MySQL para armazenar os registros de forma persistente.

MarketingPlace

O projeto MarketingPlace é um marketplace de peças de computador desenvolvido em HTML, CSS e JavaScript. Os usuários podem se cadastrar, fazer login, redefinir senha, visualizar produtos e adicionar ao carrinho.

Calculadora com gui
